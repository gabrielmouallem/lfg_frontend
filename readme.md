## Executando um Aplicativo React + Vite

Para executar um aplicativo React + Vite, você pode seguir estes passos:

1. Certifique-se de ter o Node.js e o npm (Node Package Manager) ou o Yarn instalados em seu sistema.

2. Instale as dependências do seu aplicativo. Abra o terminal e navegue até o diretório do projeto.

   ```bash
   # Se estiver usando npm
   npm install

   # Se estiver usando o Yarn
   yarn
   ```

3. Depois de instalar as dependências, você pode iniciar o servidor de desenvolvimento. No terminal, execute um dos seguintes comandos:

   ```bash
    # Se estiver usando npm
    npm start

    # Se estiver usando o Yarn
    yarn start
   ```

Esse comando irá iniciar o servidor de desenvolvimento e compilar seu aplicativo React + Vite. Ele também ficará observando alterações no código e recarregará automaticamente o aplicativo no navegador.

 - Agora basta acessar o URL: http://localhost:5173/
